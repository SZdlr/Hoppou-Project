北方　不寝番　~ Midnight Echoes


Controls
  
Arrow keys: move character  
Y: shoot  
X: bomb  
Shift: slow down character / focus fire  
Enter: ok / advance dialog  
Esc: pause / cancel  
Crtl: unlock framerate
